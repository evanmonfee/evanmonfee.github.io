HOST YOUR SITE ONLINE USING GITHUB



Host your own site for free using Github with a domain name of yourdomain.github.io. You can build elegant websites to showcase your portfolio, projects or anything else you want to share with the world.

Github serves all the sites from a personal URL tied to your Github username. The steps below detail how to create a site for windows using evanmonfee.github.io as the example.

  1. Get the HTML files for your web page ready.

  2. Create an account on Github with your username the same as the first part of your domain name i.e. 'evanmonfee' in case of evanmonfee.github.io. Alternatively you may also choose to modify the username of your existing Github account. Github allows only one site per account, so accommodate accordingly.

  3. Create a new repository in Github. Your repository name should be 'yourusername.github.io'. Make sure ‘yourusername’ in the repository name exactly matches your Github username, else it won’t work. In my case the repository name would be ‘evanmonfee.github.io’.





  4. Github uses git to host the pages. Download Github for windows here.

  5. Now open the downloaded Github application on your laptop and clone the repository you just created as shown. You will be asked to select the folder (in windows) where you want the files for your site to be stored at. For eg: D:\evanmonfee.github.io




    Once done your application would look like this:



  6. Now right click on the repository and select ‘Open in Explorer’ as shown and paste all the files for your site there (your HTML, CSS, Images, etc.). Make sure you have an index.html file.


    Once done Github automatically detects the changes made. Enter a summary for the change and select “Commit to master”. Once committed select “Publish” at the top right corner of the Github window to make your changes live.


    Github now starts syncing your changes. Once done navigate to the settings tab of your repository in your Github account online and check for the message “Your site is published at https://evanmonfee.github.io”. In some cases it might take a while for the site to get updated.

  7. That’s it. Now sit back and enjoy your custom site.
